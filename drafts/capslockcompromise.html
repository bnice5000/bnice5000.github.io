Caps Lock Compromise

Issue

I have always found the caps lock key (herein ⇪) key to be an annoyance at best and an accidental shout inducer at worst. I wanted to make the ⇪ key useful. I like doing things from the keyboard when I can. However, some tasks have no good keyboard shortcut on mac. There are solutions to this issue through software such as BetterTouchTool, Alfred, and Keyboard maestro. They all run into one major issue. There is a good chance that whatever shortcut I choose will be taken by the operating system or some already installed software package. As most software does not like to use the four modifier keys all at once, it is almost always free. Such a keyboard mash requires difficult finger gymnastics and I am not dexterous enough to execute those finger solutes.

Solution

Dr. Drang posted a solution to this issue a while ago. I thought now was the time to follow up. I decided to implement this solution and remove the scourge of the ⇪ key from my life once and for all. I modified his solution slightly so the capslock key represented

New Issue

As I leaned back to bask in the glory that was a ⇪ free life, a shout arose from the other room. Figuring this was a minor change, I had not cleared it with my significant other. She was less than pleased at the removal of her beloved ⇪ key. In her words, “You needed to give me back the ⇪ now.”

Shutting off the current solution, I tried to come up with a compromise solution. After a little testing, I though I had it. How about the ⇪ key work the new way as proposed in the Dr. Drang article, with the modification that if you used ⇪+fn it would now activate the ⇪ key to function in the usual way. She was agreeable and I implemented the solution.

New Solution

The first thing we need to do is disable the ⇪ key. I know this sounds counter intuitive but we don’t want the operating system interfering with our modifications.

Next we are going to need a little software. Download and install Seil and Karabiner. These excellent programs are written by Takayama Fumihiko. They make the process of converting the ⇪ key much easier.

Karabiner is a powerful and stable keyboard customizer for OS X. It allows you to capture and remap key presses.
