Caps Lock Compromise

Issue (First Attempt)

I have always found the caps lock key to be an annoyance at best and an accidental shout inducer at worst. I wanted to make the caps lock key useful. I like doing things from the keyboard when I can. However, some tasks have no good keyboard shortcut on mac. There are solutions to this issue through software such as BetterTouchTool, Alfred, and Keyboard maestro. They all run into one major issue. There is a good chance that whatever shortcut I choose will be taken by the operating system or some already installed software package. As most software does not like to use the four modifier keys all at once, it is almost always free. Such a keyboard mash requires difficult finger gymnastics that I am not dexterous enough to execute.

Solution (First Attempt)

Dr. Drang posted a solution to this issue a while ago. It is still mostly correct but there have been some advancements. I thought now was the time to follow up. I decided to implement his solution and remove the scourge of the caps lock key from my life once and for all. I modified his solution slightly so the caps lock key represented control+option+command+shift.

Solution (Second Attempt)

As I leaned back to bask in the glory that was a caps lock key free life, a shout arose from the other room. Figuring this was a minor change, I had not cleared it with my significant other. From the other room arose an angry hurumph. She was less than pleased at the removal of her beloved caps lock key. And I quote, “You needed to give me back the caps lock key now.”

Shutting off the current solution, I tried to come up with a compromise solution. After a little testing, I though I had it. How about if the caps lock key works as proposed in the Dr. Drang article, with the modification that if you used caps lock+fn it would now activate the caps lock key to function in the usual way. She was agreeable to the new proposed solution.

Implementation Details

The first thing we need to do is disable the caps lock key. I know this sounds counter intuitive but we don’t want the operating system interfering with our modifications.

Next we are going to need a little software. Download and install Seil and Karabiner. These excellent programs are written by Takayama Fumihiko. They make the process of converting the caps lock key much easier. Seil is a powerful key number remapper. It lets you set select keys to send new values. Karabiner is a powerful and stable keyboard customizer for OS X. It allows you to capture and remap key presses.

We need to change the key code value of the caps lock key. It is best to choose a value that is not in common use on your mac keyboard. In this case we are going to take the caps lock key with its default key value of 57 to a new value of 80 which maps to the F19 key.

With that change, the caps lock key press will now emit the F19 key code. Now to make the caps lock key press actually useful we need to add some additional behavior when the F19 key is pressed. I this case we are going to have it emit three different behaviors.

  • If pressed alone, it will still emit the F19 key code.
  • If pressed in conjunction with the fn key it will toggle the caps lock key per its normal function.
  • If pressed in conjunction with any other key it will emit the key code for the modifier keys control+option+command+shift + whatever key was pressed.
